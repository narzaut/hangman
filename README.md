## How to run

#### 1. Clone repository
```
git clone git@github.com:narzaut/hangman.git
```

#### 2. Go to cloned directory
```
cd hangman
```

#### 3. Run makefile
```
make
```

#### 4. Run executable
```
./hangman
```
